# Canva Visual Templates for Oddsify Labs

**Brand Guidelines:**
- **Background:** Black (#000000) or dark gradient
- **Primary Green:** #22c55e (EV/CLV positive)
- **Accent Red:** #ef4444 (losses/negative CLV)
- **Text:** White (#ffffff) for headers, light gray (#d1d5db) for body
- **Fonts:** Bebas Neue (headers), Inter/Roboto (body text)
- **Dimensions:** 1200x675px (Twitter landscape) or 1080x1080px (square)

---

**Bottom Banner:**
- "Process > Results"

## Template 2: Bankroll Growth Chart

**Layout:** Full chart with legend

**Chart Elements:**
- X-axis: "Number of Bets" (0, 100, 200, 300, 400, 500)
- Y-axis: "Bankroll (Units)" (starting at 100u)
- **Line 1 (Green):** CLV-Focused Growth (steady upward with variance)
- **Line 2 (Gray):** Flat Betting (flat or slight decline)
- **Shaded Areas:** Highlight drawdown periods (10-20% dips)

**Annotations:**
- Arrow pointing to variance zone: "Survived 3 drawdowns"
- Arrow pointing to growth: "+[X]% over 500 bets"
- Callout box: "1% rule adherence = long-term growth"

**Title:** "The Turtle Path: $1K → $10K in 18-24 Months"

**Bottom:**
- "5-8% success rate. Be in the 8%."
- #TestudoLegion #BankrollManagement

**File Name Format:** `bankroll-growth-scenario-[X]u.png`

## Quick Production Workflow

**For Daily Visuals (10-15 min):**
1. Open Canva → Create design → Custom 1200x675px
2. Select template from brand kit (save templates in Canva)
3. Update text/data for today's content
4. Export as PNG (high quality)
5. Save to: `~/oddsifylabs/visuals/[category]/[filename].png`
6. Post to X/Twitter with accompanying text

**Batch Creation (1x/week, 1 hour):**
- Create 5-7 poll cards for the week
- Create 2-3 BTS agent update cards
- Create 2 CLV comparison templates (fill in post-results)

## Canva Free Tier Limitations

**What Works:**
- Custom dimensions ✓
- Text overlays ✓
- Basic shapes/arrows ✓
- PNG export ✓
- Brand colors (manual hex entry) ✓

**What Requires Pro:**
- Transparent backgrounds (use remove.bg free alternative)
- Brand kit auto-save (manually save hex codes)
- Magic resize (create both 1200x675 and 1080x1080 manually)

**Free Alternative Tools:**
- **Photopea.com** — Photoshop-like, free, browser-based
- **Remove.bg** — Background removal (free tier: 50/month)
- **Excalidraw** — Hand-drawn style diagrams (free, export PNG)
