# oddsify-labs

Oddsify Labs — Sports Analytics & Betting Intelligence Landing Page

## Stack
- **Static HTML/CSS/JS** — single-file site, zero build step
- **Cloudflare Pages** — deployed via Wrangler (`wrangler.jsonc`)

## Sections
- Hero with animated particle background
- Products (6-card grid)
- How It Works (3-step process)
- Stats (animated counters)
- Pricing (3 tiers)
- Testimonials
- Contact form
- Footer

## Design System
- Dark theme (`#0a0e1a` background)
- Blue-purple-cyan gradient accents
- Glassmorphism cards
- Scroll-triggered animations
- Responsive (mobile-first)

## Deployment
```bash
npx wrangler pages deploy .
```
